# 介绍

使用 Swift 语言编写的 Material Design 风格的 iOS 流媒体音乐播放器，简称 MMP。基于 [DOUAudioStreamer-Swift](https://github.com/johnlui/DOUAudioStreamer-Swift)。

## Features

- [x] 完全流媒体播放
- [x] 锁屏信息展示及控制
- [x] Apple Watch 展示及控制
- [x] 界面简洁优雅
